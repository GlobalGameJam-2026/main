# 가면 전환 효과 추가 가이드

## 🎨 화면 페이드/플래시 효과 추가하기

### 1단계: Canvas 및 Fade Image 설정

1. **Canvas 생성**
   - Hierarchy 우클릭 → UI → Canvas
   - 이름: `UI_Canvas`

2. **Canvas 설정**
   - Render Mode: Screen Space - Overlay
   - (이미 기본값으로 설정되어 있음)

3. **Fade Image 생성**
   - Canvas 우클릭 → UI → Image
   - 이름: `FadeImage`

4. **FadeImage 설정**
   - Inspector에서:
     - **Anchor**: Stretch (좌우상하 모두 늘림)
       - Anchor Presets 클릭 → Alt+Shift 누른 채로 우측 하단 클릭
     - **Color**: 흰색 (255, 255, 255, 0) - **Alpha는 0으로!**
     - **Raycast Target**: ✗ (체크 해제) - 클릭 방지

---

### 2단계: MaskTransitionEffect 컴포넌트 추가

1. **Canvas 선택**
   - Hierarchy에서 `UI_Canvas` 클릭

2. **컴포넌트 추가**
   - Inspector → Add Component
   - `MaskTransitionEffect` 검색 후 추가

3. **설정**
   - **Fade Image**: FadeImage를 드래그해서 할당
   - **Fade Duration**: 0.2 (기본값, 조절 가능)
   - **Use Mask Specific Colors**: ✓ (체크) - 가면별 다른 색상
   - **Black Mask Flash Color**: 흰색 (255, 255, 255) - 검은 가면으로 전환 시
   - **White Mask Flash Color**: 검은색 (0, 0, 0) - 흰 가면으로 전환 시
   - **Flash Intensity**: 0.5 (조절 가능, 0~1)
   - **Spawn Particles On Transition**: ✓ (파티클 사용 시)

---

## ✨ 파티클 효과 추가하기 (선택사항)

### 3단계: 파티클 시스템 생성

1. **Hierarchy에서 Particle System 생성**
   - 우클릭 → Effects → Particle System
   - 이름: `MaskTransitionParticles`

2. **위치 설정**
   - Transform Position: (0, 0, 0)
   - Canvas의 자식으로 만들기 (드래그해서 Canvas 안으로)

3. **Particle System 설정**

#### Main 모듈  
- **Duration**: 0.3
- **Looping**: ✗ (체크 해제)
- **Start Lifetime**: 0.5
- **Start Speed**: 5
- **Start Size**: 0.1
- **Start Color**: 흰색 또는 그라데이션
- **Max Particles**: 50
- **Play On Awake**: ✗ (체크 해제) - 중요!

#### Emission 모듈
- **Rate over Time**: 0
- **Bursts**: 
  - Time: 0
  - Count: 30

#### Shape 모듈
- **Shape**: Sphere
- **Radius**: 0.5

#### Color over Lifetime (선택사항)
- 그라데이션으로 페이드 아웃 효과

#### Size over Lifetime (선택사항)
- 시간에 따라 크기 감소

---

### 4단계: 파티클을 MaskTransitionEffect에 연결

1. **Canvas 선택** (UI_Canvas)

2. **MaskTransitionEffect 컴포넌트 찾기**

3. **Transition Particles 할당**
   - `MaskTransitionParticles`를 드래그해서 할당

---

## 🎮 테스트하기

1. **Play 버튼** 클릭

2. **Left Shift** 눌러서 가면 전환

3. **확인 사항**
   - ✓ 검은 가면으로 전환 시 → 화면이 하얗게 번쩍임
   - ✓ 흰 가면으로 전환 시 → 화면이 검게 번쩍임
   - ✓ 부드럽게 페이드 인/아웃
   - ✓ 파티클이 터짐 (설정했다면)

---

## ⚙️ 효과 조절하기

### 더 강한 플래시 효과
- **Flash Intensity**: 0.8~1.0으로 증가

### 더 긴 페이드 효과
- **Fade Duration**: 0.3~0.5로 증가

### 검은색 페이드 (플래시 대신)
- **Use Flash Effect**: ✗ (체크 해제)
- 또는 **Flash Color**: 검은색 (0, 0, 0)

### 파티클 색상 변경
- Particle System → Main → Start Color
- 보라색, 파란색 등 감정 색상으로 변경 가능
