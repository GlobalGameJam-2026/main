# 문제 해결 가이드 - 타이틀 화면

## 🐛 게임 종료 버튼이 작동하지 않는 문제

**증상**: 게임 종료 버튼을 눌러도 스테이지 선택 화면으로 이동합니다.

**원인**: QuitButton에 QuitButton 컴포넌트가 없거나, StartButton의 OnClick 이벤트가 그대로 복사되어 있습니다.

### 해결 방법:

1. **Hierarchy에서 QuitButton 선택**

2. **Inspector 확인**:
   - ✅ **QuitButton 컴포넌트**가 있는지 확인
   - 없다면: Add Component → `QuitButton`

3. **Button 컴포넌트의 OnClick() 이벤트 정리**:
   - Button 컴포넌트 → **OnClick()** 섹션
   - 리스트에 항목이 있다면 **모두 삭제** (- 버튼 클릭)
   - QuitButton 컴포넌트가 자동으로 처리하므로 OnClick에는 **아무것도 없어야 합니다**

4. **테스트**:
   - Play 모드에서 게임 종료 버튼 클릭
   - Unity 에디터에서는 Play 모드가 종료됩니다
   - 빌드된 게임에서는 앱이 종료됩니다

## 🎨 스테이지 버튼 색상 문제

**증상**: 스테이지 버튼이 모두 하얀색으로 표시됩니다.

**원인**: Button의 Image 컴포넌트 색상이 흰색으로 설정되어 있습니다.

### 해결 방법 1: 수동 변경

1. **Stage1Button 선택**

2. **Image 컴포넌트에서**:
   - **Color**: 원하는 색상으로 변경
   - 예: 파란색 (R:100, G:150, B:255, A:255)

3. **Stage2Button도 같은 방식으로 변경**

### 해결 방법 2: 스크립트 자동 처리 (권장)

StageButton.cs 스크립트가 업데이트되어 자동으로 색상을 변경합니다.

1. **Unity 재시작** 또는 **Scripts 폴더 우클릭 → Reimport**

2. **Play 모드 실행**:
   - Stage 1: 잠금 해제 → 흰색 (또는 설정한 unlockedColor)
   - Stage 2: 잠금 상태 → 회색 (또는 설정한 lockedColor)

3. **색상 커스터마이징**:
   - Stage1Button 선택 → StageButton 컴포넌트
   - **Unlocked Color**: 잠금 해제 시 색상 (기본: 흰색)
   - **Locked Color**: 잠금 시 색상 (기본: 회색)

## 🔧 추가 확인 사항

### QuitButton이 보이지 않는 경우

1. **Canvas Group Alpha 확인**:
   - QuitButton → Canvas Group
   - Alpha가 0이면 투명합니다
   - 임시로 1로 설정해서 확인

2. **TitleScreenManager 연결 확인**:
   - TitleScreenManager 선택
   - **Quit Button Canvas Group**: QuitButton의 Canvas Group 드래그

### 버튼 클릭이 안 되는 경우

1. **EventSystem 확인**:
   - Hierarchy에 EventSystem이 있는지 확인

2. **Canvas Group 설정**:
   - Interactable: ✓
   - Blocks Raycasts: ✓

## ✅ 정상 작동 확인

### 게임 종료 버튼:
- Unity 에디터: Play 모드 종료
- 빌드된 게임: 앱 종료
- Console에 "Quitting game..." 메시지 출력

### 스테이지 버튼:
- Stage 1: 항상 활성화 (흰색 또는 설정한 색상)
- Stage 2: Stage 1 클리어 전까지 비활성화 (회색)
- Stage 1 클리어 후: Stage 2 활성화 (색상 변경)
